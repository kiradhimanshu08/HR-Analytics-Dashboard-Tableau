# HR-Analytics-Dashboard-Tableau
HR analytics dashboard analyzing employee attrition, job satisfaction, and workforce demographics using Tableau.


## Tools Used
Tableau
Excel
Data Visualization

## Key Insights
* Total Employees: 1470
* Attrition Count: 237
* Attrition Rate: 16.12%
* Highest attrition occurs in the Sales department
* Employees aged 25–34 show the highest attrition
* Job satisfaction varies significantly across job roles
